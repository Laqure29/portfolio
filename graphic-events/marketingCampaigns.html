Event Branding

Annual Design Awards – Gwalior

For the Annual Design Awards held in Gwalior, I executed end-to-end branding for the event. This included the main gate and entrance design, ID cards, lanyards, event passes, a checkered branding backdrop, and an interactive selfie point. The entire visual identity was developed using the institute’s house colors, enhanced with bold geometric shapes to create a vibrant, modern, and cohesive look throughout the event space.
